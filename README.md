# Agro Forte, Futuro Sustentável: Equilíbrio entre Produção e Meio Ambiente

## Descrição
Este projeto é uma Landing Page informativa desenvolvida para conscientizar o público sobre o papel do agronegócio moderno em sintonia com a preservação ambiental. O objetivo é demonstrar visual e textualmente como a tecnologia no campo (como agricultura de precisão) permite alcançar alta produtividade sem abrir mão da sustentabilidade e do respeito aos recursos naturais.

O site foi estruturado com foco em **acessibilidade web (A11y)**, garantindo uma navegação semântica para leitores de tela, e **design responsivo**, adaptando-se perfeitamente a computadores, tablets e celulares.

## Tecnologias utilizadas
- **HTML5:** Estruturação semântica e acessível (uso de tags como `<header>`, `<main>`, `<section>`, `<article>`).
- **CSS3:** Estilização baseada em uma identidade visual coesa (tons de verde e terra), utilizando layouts modernos com **Flexbox** e **CSS Grid**.
- **JavaScript:** Implementação de interações funcionais e dinâmicas com o usuário.
